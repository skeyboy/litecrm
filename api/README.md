## Description

[litecrm](https://github.com/litecrm/litecrm) 基于nestjs+ant design pro 开源CRM系统

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
## 修改配置信息
```bash

cp .env.example .env

```
